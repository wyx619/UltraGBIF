# Package index

## All functions

- [`check_occ_name()`](https://wyx619.github.io/UltraGBIF/reference/check_occ_name.md)
  : Use local or online taxon resolution service to check accepted taxon
  names and update synonyms at once
- [`generate_collection_mark()`](https://wyx619.github.io/UltraGBIF/reference/generate_collection_mark.md)
  : Generate unique collection event
- [`get_collectors_name()`](https://wyx619.github.io/UltraGBIF/reference/get_collectors_name.md)
  : Extract last name of the main collector (internal)
- [`import_records()`](https://wyx619.github.io/UltraGBIF/reference/import_records.md)
  : Import GBIF occurrence records
- [`map_records()`](https://wyx619.github.io/UltraGBIF/reference/map_records.md)
  : Visualize refined records on a dynamic interactive map
- [`prepare_collectors_dictionary()`](https://wyx619.github.io/UltraGBIF/reference/prepare_collectors_dictionary.md)
  : Prepare a list with surname of the main collector
- [`refine_records()`](https://wyx619.github.io/UltraGBIF/reference/refine_records.md)
  : Restore duplicate records from voucher, validate coordinates and
  extract their World Geographical Scheme for Recording Plant
  Distributions
- [`set_digital_voucher()`](https://wyx619.github.io/UltraGBIF/reference/set_digital_voucher.md)
  : Set master digital voucher
- [`standardize_name()`](https://wyx619.github.io/UltraGBIF/reference/standardize_name.md)
  : Standardize scientific name (internal)
- [`usecores()`](https://wyx619.github.io/UltraGBIF/reference/usecores.md)
  : Detect threads and optimize them (internal)
- [`wcvp_check_name()`](https://wyx619.github.io/UltraGBIF/reference/wcvp_check_name.md)
  : Resolve taxons with WCVP database (internal)
